# 微信公众号平台特征参考

## 用户画像

| 属性 | 描述 |
|------|------|
| **年龄** | 25-45岁为主 |
| **职业** | 职场人士、创业者、自由职业 |
| **阅读场景** | 上下班通勤、午休、睡前 |
| **阅读习惯** | 碎片化阅读，快速浏览 |
| **内容偏好** | 实用性强、有深度、能引发共鸣 |

## 内容调性

| 内容类型 | 风格要求 |
|----------|----------|
| **技术类** | 专业、准确、实战案例 |
| **职场类** | 真诚、有洞察、方法论 |
| **生活类** | 温暖、有情感、正能量 |
| **观点类** | 观点明确、论据充分、逻辑清晰 |

## 阅读行为特点

1. **首屏决定去留** - 前3秒决定是否继续阅读
2. **标题点击率关键** - 好标题占70%成功因素
3. **视觉疲劳快速** - 需要图片、分隔线缓解
4. **互动意愿强** - 喜欢、在看、分享意愿高
5. **收藏率反映价值** - 实用内容收藏率更高
